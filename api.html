API и интеграция

Прямое H2H-соединение без посредников. Документация и примеры запросов.

Базовые принципы

  • 🔹 REST API over HTTPS
  • 🔹 Аутентификация по API-ключу (в заголовке X-API-Key)
  • 🔹 Все ответы в формате JSON
  • 🔹 Поддержка идемпотентности (заголовок Idempotency-Key)
